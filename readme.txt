Website ini adalah sebuah platform reservasi (pemesanan kamar) untuk hotel bernama Star Hotel. Didalamnya terdapat semua file utama dan 2 folder utama yaitu assets yang isinya merupakan resource berupa styling seperti bootstrap (css), javascript dan image, kemudian folder dengan nama function berisi 3 file yang digunakan untuk menyimpan script utama seperti koneksi database, insert database, ambil data dari database.
